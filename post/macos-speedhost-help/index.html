Introduction of SpeedHost for macOS

SpeedHost is a proxy utility which used Network Extension Technology.

Features

  • Support to add SS Server.
  • Support Auto mode and Global mode
  • Support Scan QR Code and Generate QR Code.
  • No registration and login, easy to use.
