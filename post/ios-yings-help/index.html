Introduction of Yings for iOS

Overview

Yings is a proxy utility which used shadowsocks, shadowsocks is a secure socks5 proxy, designed to protect your internet traffic.

Features

  1. Super Fast and secured with industry level encryption algorithm.
  2. Suports auto mode and global mode
  3. With eight built-in proxy servers, more proxy servers will be available soon.
  4. No registration and login, easy to use.

The difference of Auto Mode and Global Mode

Auto Mode: You know some websites is blocked in some country. So when you browse these websites, i think “Yings” will help you. When you browse normal websites, “Yings” will do nothing.

Global Mode: “Yings” will proxy all your network traffic.
