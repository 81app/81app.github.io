Secnettool Introduction

SecNetTool SecNetTool is an IPsec VPN Client, which encrypts your network traffic, so that nobody between you and the VPN server can eavesdrop on your data as it travels via the Internet. This is especially useful when using unsecured networks, e.g. at coffee shops, airports or hotel rooms.
